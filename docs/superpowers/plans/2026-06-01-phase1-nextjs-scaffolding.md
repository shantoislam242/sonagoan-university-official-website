# Phase 1 — Next.js Scaffolding + Asset Migration + main.js Inspection

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up an empty Next.js 15 App Router project at `su-website-next/`, copy the static template assets into `public/assets/`, load the template's vendor CSS/JS in the root layout, render a placeholder home page, and document `main.js` behavior — without modifying any vendor or template file.

**Architecture:** Manual Next.js + TypeScript setup (no `create-next-app` boilerplate). Vendor CSS via `<link>` in `<head>`, vendor JS via `<Script>` at the end of `<body>`, jQuery `beforeInteractive`, all plugins + `main.js` `afterInteractive`. Pages and the root layout stay Server Components. `main.js` is read and documented but not modified.

**Tech Stack:** Next.js 15.x, React 19.x, TypeScript 5.x. No additional runtime deps in this phase (per the vendor dependency policy in the spec — static assets are the default).

**Spec reference:** `docs/superpowers/specs/2026-06-01-su-website-nextjs-conversion-design.md` (FROZEN). This plan implements the "Phase 1 — Scaffolding + assets + inspection" section.

```ts
import type { Metadata } from 'next';

const siteName = 'SU Website';
// Replace before production deploy. Override via NEXT_PUBLIC_SITE_URL when set.
const siteUrl =
  process.env.NEXT_PUBLIC_SITE_URL ?? 'https://example.com';

export interface BuildMetadataInput {
  title: string;
  description: string;
  path: string;
  image?: string;
}

export function buildMetadata(input: BuildMetadataInput): Metadata {
  const url = `${siteUrl}${input.path}`;
  const image = input.image ?? '/assets/images/og-default.jpg';
  return {
    title: `${input.title} | ${siteName}`,
    description: input.description,
    openGraph: {
      title: input.title,
      description: input.description,
      url,
      siteName,
      images: [{ url: image }],
      type: 'website',
    },
    twitter: {
      card: 'summary_large_image',
      title: input.title,
      description: input.description,
      images: [image],
    },
  };
}
```
